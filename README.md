# GlicoSIM - Sistema de controle glicêmico entre outras ferramentas

Com o GlicoSIM você está no controle da sua diabetes seja ela tipo 1 ou tipo 2.

Tenha avisos regulares, dicas, exercícios e várias outras ferramentas para te ajudar a controlar a glicemia.

https://glicosim.vercel.app

## Rodando localmente

**Prerequisites:**  Node.js


1. Instalando dependências:
   `npm install`
2. Set o `GEMINI_API_KEY` in [.env.local](.env.local) no seu arquivo
3. Rode o comando:
   `npm run dev`
